About

OnlineTuner.co is a free tuner for your guitar. It uses only web technologies (Web Audio API and canvas).

OnlineTuner is powered by beautiful open source technologies. Enjoy what lots of people have done with passion.

How does it works

OnlineTuner.co use new Web Audio API to capture sound, applies BlackMan window on input signal, computes the FFT through dsp.js and applies microphone response correction.
